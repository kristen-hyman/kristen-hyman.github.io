Summary: Latitude vs. Temperature


The purpose of this project was to analyze how weather changes as you get to the equator. To accomplish this analysis, we first pulled data from the OpenWeatherMap API to assemble a dataset of over 500 cities.


This graph shows the trending between max temperature and latitude.
